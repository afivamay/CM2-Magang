# Sistem Pendaftaran Magang Mahasiswa

Program ini dibuat untuk membantu panitia magang mencatat dan mengelola data pendaftaran magang mahasiswa.

## Fitur Program

1. **Tambah Data Magang**
   - Memasukkan data mahasiswa yang mendaftar magang.
   - Data yang dicatat:
     - Nama Mahasiswa
     - NIM
     - Program Studi
     - Perusahaan Tujuan Magang
     - Semester Pengambilan Magang (hanya 6 atau 7)
     - Status Magang (hanya Diterima, Menunggu, atau Ditolak)
   - Program melakukan validasi untuk **semester** dan **status**.

2. **Tampilkan Semua Pendaftar**
   - Menampilkan semua data pendaftar magang dalam bentuk tabel.

3. **Cari Pendaftar Berdasarkan Program Studi**
   - Menampilkan data pendaftar berdasarkan program studi yang dicari.

4. **Hitung Status Magang**
   - Menghitung jumlah pendaftar berdasarkan status:
     - Diterima
     - Menunggu
     - Ditolak

5. **Keluar**
   - Keluar dari program.
